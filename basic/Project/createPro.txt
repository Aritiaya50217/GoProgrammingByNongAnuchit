วิธีการสร้าง Project ใน golang
- สร้างโฟลเดอร์ขึ้นมา 
- สร้างไฟล์นามสกุล .go
- ไฟล์เริ่มต้นของโปรเจค ต้องเริ่มด้วย package main และมี func main()
- สร้าง go mod โดย พิมพ์ go mod init(ชื่อโปรเจคของเรา)
- จะได้ไฟล์ go.mod คือไฟล์ module
- สามารถแก้ไขชื่อ module ภายหลังได้ เช่น module igapp เปลี่ยนเป็น module github.com/Artit50217/igapp

การแยก Package
- สร้างโฟลเดอร์ที่ชื่อเดียวกับ package
- เมื่อมีการสร้าง package ใหม่ ไฟล์แรกที่ควร create คือ ไฟล์แรกที่เป็นชื่อเดียวกันกับ Package
- ไปที่ไฟล์หลัก (main.go) เพิ่ม import ("ชื่อโปรเจคของเรา/โฟลเดอร์ที่เราใช้เก็บ packageใหม่")
- เวลาเรียกใช้งาน เป็นดังนี้ 
func main() {user.Info("Oil", "Golang", 23) }โดย user คือ packageใหม่ที่เราสร้าง เอามา.กับชื่อ func ที่อยู่ในไฟล์หลัก (main.go) func Info มาจากไฟล์ user.go ในไฟล์ packageใหม่ที่เราสร้าง *** ถ้าตัวพิมพ์เล็ก (info) จะเป็น Private ถ้าพิมพ์ใหญ่ (Info) จะเป็น Public เวลาเรียกใช้งานควรเป็น พิมพ์ใหญ่เพราะใช้ประกาศภายนอก
